POS Network Printer


Odoo Extension Allowing The User To Print Point Of Sales Order Receipts via ESC/POS Network Printer. No need of a PosBox.

This module uses the QZ Tray client for sending direct print commands to the ESC/POS printer:

  • Installation Link: ( https://qz.io/download/ )
  • Demo Link: ( https://demo.qz.io/ )

Salient Features

  • User can print the order receipts via the ESC/POS Network Printer. There is no need to use the PosBox.
  • User can configure the network printer's name in the POS Configuration page.
  • Qz Tray client must be installed for sending direct print commands to the ESC/POS printer.
  • User can check/refresh the printer connection directly from the POS Interface.

POS Network Printer

Configuring Network Printer

User can specify the ESC/POS Network Printer name in the Point Of Sale configuration form under the IotBox / Hardware Proxy section.

Make sure the Printer's name is exactly the same as the name of the printer is used by the QZ Tray client to make connection and send the print commands.

POS Network Printer

ESC/POS Network Printer Connection

POS automatically connects to the configured printer when it is loaded. There is a connection notification icon that the user can check for the connection status. User can also click on the connection notification icon if he wishes to refresh the connection.

Following are the possible connection statuses:

Not-Connected

This means that the POS was unable to connect to the specified printer. Make sure you have the Qz Tray client setup and its running .

Connected To Qz Tray

This means that the POS was able to connect to the Qz Tray client on your system and now needs to connect to the configured ESC/POS printer.

Connected To The Specified Printer

This means that the POS has successfully established the connection with the configured printer via the Qz Tray Client.

Printing the Receipt

POS automatically sends the print command to the ESC/POS Network Printer when the order is validated.

User can print the receipts again from the Receipt Screen if he wishes to.
